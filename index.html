This is a demo

git is version control software on your computer.

We COMMIT our code version to git on our computer.

github.com is a cloud service that works with git to help us collaborate with other coders.

We PUSH our code to github.com to collaborate with other coders.

We DEPLOY our projects to a server so that the world can view them in a browser on a computer, tablet or phone.

In the bootcamp we DEPLOY to the bootcamp-coders.cnm.edu server

DEPLOY and PUSH are DIFFERENT!!!
